# 信贷场景中什么是评分卡？

  1、以分数的形式来衡量风险几率的一种手段
  
  2、对未来一段时间内违约/逾期/失联概率的预测
  
  3、通常评分越高越安全
  
  4、根据使用场景分为反欺诈评分卡、申请评分卡、行为评分卡、催收评分卡
